Instruções para rodar o projeto

> 5 módulos:

Execução do front:
	> npm install (para instalar dependências);
	> npm run dev

Execução travel agency:
	> python agencia viagens.py

Execução airline:
	> python companhia aérea.py

Execução hotel chain:
	> na pasta ./hotel-chain/bin
	> hotel-chain.exe

Execução car rental:
	> na pasta ./car-rental/bin
	> car-rental.exe

> Estando todos executando, basta entrar com dados no front para solicitar a viagem. Dados abaixo das passagens disponíveis:
